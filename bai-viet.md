## Tổng quan

Bài viết dùng cho blog/tin tức/hướng dẫn. Bài viết hỗ trợ phân loại theo **danh mục bài viết** và tối ưu SEO.

## Thuộc tính cơ bản

- Tiêu đề
- Tác giả (tùy chọn)
- Nội dung (Markdown/HTML/Rich text tùy hệ thống)
- Ảnh đại diện
- Danh mục bài viết (một hoặc nhiều)
- Tags (tùy chọn)
- Trạng thái: nháp / lên lịch / đã xuất bản / ẩn

## SEO

- Slug
- Meta title / meta description
- OG image (nếu có)

## Quy trình xuất bản (tham khảo)

1. Tạo bài viết nháp
2. Gắn danh mục + tối ưu SEO
3. Xem trước (preview)
4. Xuất bản hoặc lên lịch
